what is html?

Hypertext markup language is a computer language which can be used to create webpages and applications.

What is a basic structure of an HTML document?

It consists of 5 basic structure, The priority is as follows:

  • DOCTYPE:

    By the doctype determines which version of html is used and written.

  • Html:

    Html notifies the browser of the html page format and indicates which global language is used.

  • Head:

    It is the part that is not displayed in the web browser when the page is loaded.in the other words it is "behind the scenes" and including many data such as title,css link and so on.

  • Body:

    The body part has all actual content such as text,tables,videos,links and etc.

  • Title:

    By the title,title displayed in the toolbar browser.
